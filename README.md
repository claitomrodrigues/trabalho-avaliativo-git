# 🚀 Atividade Prática: Fluxo de Trabalho com Git

Este repositório contém o desenvolvimento de uma atividade prática focada no domínio do **Git**, explorando desde configurações iniciais até a resolução de conflitos em ambientes de ramificação (branches).

## 🎯 Objetivos do Projeto
O projeto simula um fluxo real de trabalho, onde diferentes versões de um documento (Introdução, Desenvolvimento e Conclusão) são editadas simultaneamente em branches distintas, exigindo integração manual e gestão de histórico.

## 🛠️ Tecnologias e Comandos Utilizados
- **Configuração:** `git config` para identidade do autor.
- **Ramificação:** Uso de `git branch` e `git checkout` para isolar o trabalho em `desenvolvimento` e `revisao`.
- **Sincronização:** `git merge` para integração de funcionalidades.
- **Resolução de Conflitos:** Tratamento manual de divergências entre arquivos.
- **Visualização:** Logs avançados com `git log --graph --oneline --all`.

## 📁 Estrutura de Arquivos
- `introducao.txt`, `desenvolvimento.txt`, `conclusao.txt`: Arquivos base do projeto.
- `comandos.txt`: Registro de todos os passos executados no terminal.
- `*.png`: Prints das etapas de log para comprovação do histórico de commits.

## 🧠 Aprendizados Chave
> "O Git não é apenas uma ferramenta de backup, mas uma máquina do tempo e um espaço de colaboração."
Durante esta atividade, foi possível consolidar o entendimento sobre como o Git gerencia o grafo de commits e a importância de mensagens de commit claras para a manutenção do código.
